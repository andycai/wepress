# Wepress

Wepress is the lightweight open-source headless CMS wrote in Rust.

Rust + Axum + Alpine.js

Modules include: 

- Site
- Content
    - Category
    - Post
    - Page
    - Media
- Setting
    - Group
    - GroupMember
    - User
    - Config
- Activity
    - Activity
    - Club
    - ClubMember
